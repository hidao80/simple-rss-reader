A simple RSS reader.

## Features

- [x] Subscribe to multiple subscription URLs
- [ ] Automatic updates
- [x] Storing URLs for RSS without server using localStorage
- [ ] Generation of text for SNS sharing
- [ ] Batch export/import of RSS URLs during registration
